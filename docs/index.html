Philosophy

Native, not Electron

Pure SwiftUI. Launches instantly, uses minimal resources, and feels like it belongs on your Mac.

120fps, not loading spinners

Butter-smooth scrolling through even the longest documents. No jank, no lag, no waiting.

Calm, not noisy

Chrome-less window. Distraction-free reading. Keyboard-first. Just you and the document.

Built for markdown that matters

Native SwiftUI rendering

Every heading, table, and code block rendered natively by SwiftUI and TextKit 2. Mermaid diagrams get lightweight embedded web views — everything else is pure platform.

Mermaid diagrams

Embedded, interactive diagrams. Flowcharts, sequence diagrams, state machines — click to focus, rendered in place.

Solarized theming

Dark, light, and auto modes. The Solarized palette designed for long reading sessions and code-heavy documents.

LaTeX math

Native rendering of block and inline equations. Integrals, summations, Greek letters — all typeset beautifully without a web engine.

Syntax highlighting

Tree-sitter highlighting across 16 languages. Code blocks with language labels, themed backgrounds, and click-to-copy.

Distraction-free, keyboard-first

No toolbar clutter. Chrome-less windows. Navigate, scroll, and switch themes without reaching for the mouse.

Real math. Natively rendered.

Integrals, summations, fractions, and Greek letters — typeset with native text rendering. No MathJax, no web views, no loading delays.

Two palettes. One philosophy.

Solarized Dark for focus. Solarized Light for daylight. Both precision-tuned for readability.

Get started

macOS 14.0 or later.

Homebrew 
brew tap jud/mkdn && brew install --cask mkdn
Build from source 
git clone https://github.com/jud/mkdn.git && cd mkdn && swift run mkdn
